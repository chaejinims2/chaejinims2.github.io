cat /proc/mdstat 에서 알 수 있는 정보
=============================

요약
--
- 제목: **소프트웨어 RAID 상태 확인**
- 설명: MD(소프트웨어 RAID) 장치 상태 및 미사용 장치 확인

로그 주요 출력(일부)
------------

.. code-block:: text

  cherry@rhombus:~$ cat /proc/mdstat
  Personalities : 
  unused devices: <none>

확인 포인트(체크리스트)
-------------
- 출력에서 **핵심 컬럼 이름**(헤더)을 먼저 읽는다.
- 숫자/단위(%, bytes, blocks, port 등)를 보고 **무엇을 의미하는지** 말로 설명한다.
- 설정 파일이라면(예: fstab) “무엇을 어디에 어떻게 마운트하는지”를 한 문장으로 요약한다.
