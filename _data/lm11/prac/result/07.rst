du -sh * 에서 알 수 있는 정보
=====================

요약
--
- 제목: **디렉터리 총 사용량**
- 설명: 어느 디렉터리가 많이 쓰는지 확인

로그 주요 출력(일부)
------------

.. code-block:: text

  cherry@rhombus:~$ du -sh *
  12K	backup.tar
  4.0K	lm11-prac-add.sh
  8.0K	lm11-prac-result.sh
  8.0K	lm11-prac.json
  4.0K	lm11-prac.sh
  40K	log
  4.0K	result
  16K	tmp

확인 포인트(체크리스트)
-------------
- 출력에서 **핵심 컬럼 이름**(헤더)을 먼저 읽는다.
- 숫자/단위(%, bytes, blocks, port 등)를 보고 **무엇을 의미하는지** 말로 설명한다.
- 설정 파일이라면(예: fstab) “무엇을 어디에 어떻게 마운트하는지”를 한 문장으로 요약한다.
