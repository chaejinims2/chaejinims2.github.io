lsblk 에서 알 수 있는 정보
==================

요약
--
- 제목: **블록 장치 트리 목록**
- 설명: 블록 장치·파티션을 트리 형태로 목록 출력 (파일시스템·UUID 미포함)

로그에서 바로 읽히는 포인트
---------------
- mmcblk 관련 행 예시: ```-mmcblk1p5  179:5    0    40G  0 part /home/***REDACTED_USER***/workspace/chaejinims2.github.io/.git/hooks``

확인 포인트(체크리스트)
-------------
- 출력에서 **핵심 컬럼 이름**(헤더)을 먼저 읽는다.
- 숫자/단위(%, bytes, blocks, port 등)를 보고 **무엇을 의미하는지** 말로 설명한다.
- 설정 파일이라면(예: fstab) “무엇을 어디에 어떻게 마운트하는지”를 한 문장으로 요약한다.
