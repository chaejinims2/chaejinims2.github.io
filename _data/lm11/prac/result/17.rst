kill -l 에서 알 수 있는 정보
====================

요약
--
- 제목: **시그널 목록 출력**
- 설명: 프로세스에 보낼 수 있는 시그널 번호·이름 목록 확인

로그에서 바로 읽히는 포인트
---------------
- 시그널 번호/이름 매핑 확인 가능 (예: SIGTERM=15, SIGKILL=9)

확인 포인트(체크리스트)
-------------
- 출력에서 **핵심 컬럼 이름**(헤더)을 먼저 읽는다.
- 숫자/단위(%, bytes, blocks, port 등)를 보고 **무엇을 의미하는지** 말로 설명한다.
- 설정 파일이라면(예: fstab) “무엇을 어디에 어떻게 마운트하는지”를 한 문장으로 요약한다.
