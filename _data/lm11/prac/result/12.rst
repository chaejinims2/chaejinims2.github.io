whereis df 에서 알 수 있는 정보
=======================

요약
--
- 제목: **df 바이너리·매뉴얼 위치**
- 설명: df 실행 파일, 소스, 매뉴얼 페이지 경로 검색

로그 주요 출력(일부)
------------

.. code-block:: text

  cherry@rhombus:~$ whereis df
  df: /usr/bin/df /usr/share/man/man1/df.1.gz

확인 포인트(체크리스트)
-------------
- 출력에서 **핵심 컬럼 이름**(헤더)을 먼저 읽는다.
- 숫자/단위(%, bytes, blocks, port 등)를 보고 **무엇을 의미하는지** 말로 설명한다.
- 설정 파일이라면(예: fstab) “무엇을 어디에 어떻게 마운트하는지”를 한 문장으로 요약한다.
