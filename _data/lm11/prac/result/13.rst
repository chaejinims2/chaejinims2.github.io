which df 에서 알 수 있는 정보
=====================

요약
--
- 제목: **df 실행 파일 경로**
- 설명: PATH에서 df 명령이 실제로 실행되는 경로 출력

로그 주요 출력(일부)
------------

.. code-block:: text

  cherry@rhombus:~$ which df
  /usr/bin/df

확인 포인트(체크리스트)
-------------
- 출력에서 **핵심 컬럼 이름**(헤더)을 먼저 읽는다.
- 숫자/단위(%, bytes, blocks, port 등)를 보고 **무엇을 의미하는지** 말로 설명한다.
- 설정 파일이라면(예: fstab) “무엇을 어디에 어떻게 마운트하는지”를 한 문장으로 요약한다.
