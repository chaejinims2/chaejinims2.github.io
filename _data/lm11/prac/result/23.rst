echo $PATH 에서 알 수 있는 정보
=======================

요약
--
- 제목: **PATH 환경 변수 출력**
- 설명: 실행 파일을 찾는 검색 경로(PATH) 확인

로그 주요 출력(일부)
------------

.. code-block:: text

  cherry@rhombus:~$ echo $PATH
  /home/***REDACTED_USER***/.local/bin:/home/***REDACTED_USER***/bin:/home/***REDACTED_USER***/.cursor-server/cli/servers/Stable-60faf7b51077ed1df1db718157bbfed740d2e160/server/bin/remote-cli:/home/***REDACTED_USER***/.local/bin:/home/***REDACTED_USER***/bin:/home/***REDACTED_USER***/.nvm/versions/node/v18.20.8/bin:/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin:/usr/games:/usr/local/games:/snap/bin:/home/***REDACTED_USER***/bin:/home/***REDACTED_USER***/bin

확인 포인트(체크리스트)
-------------
- 출력에서 **핵심 컬럼 이름**(헤더)을 먼저 읽는다.
- 숫자/단위(%, bytes, blocks, port 등)를 보고 **무엇을 의미하는지** 말로 설명한다.
- 설정 파일이라면(예: fstab) “무엇을 어디에 어떻게 마운트하는지”를 한 문장으로 요약한다.
