df 에서 알 수 있는 정보
===============

요약
--
- 제목: **disk free**
- 설명: 마운트된 파일시스템의 디스크 사용량·여유 공간(기본 단위)

로그에서 바로 읽히는 포인트
---------------
- 루트(/) 사용량 행: ``/dev/mmcblk1p5  41000364 26292792  12592656  68% /``

로그 주요 출력(일부)
------------

.. code-block:: text

  cherry@rhombus:~$ df
  Filesystem     1K-blocks     Used Available Use% Mounted on
  /dev/mmcblk1p5  41000364 26292792  12592656  68% /
  none                  64        0        64   0% /dev
  tmpfs             794036     2100    791936   1% /run
  tmpfs               5120        4      5116   1% /run/lock
  tmpfs             794032       96    793936   1% /run/user/1000
  efivarfs             192       39       149  21% /sys/firmware/efi/efivars
  /dev/mmcblk1p1     98304    57244     41060  59% /boot/efi
  /dev/mmcblk2p1   4894364  4894364         0 100% /media/cherry/Ubuntu 22.04.4 LTS amd64
  /dev/mmcblk2p4  56851280     1552  53929436   1% /media/cherry/writable

확인 포인트(체크리스트)
-------------
- 출력에서 **핵심 컬럼 이름**(헤더)을 먼저 읽는다.
- 숫자/단위(%, bytes, blocks, port 등)를 보고 **무엇을 의미하는지** 말로 설명한다.
- 설정 파일이라면(예: fstab) “무엇을 어디에 어떻게 마운트하는지”를 한 문장으로 요약한다.
