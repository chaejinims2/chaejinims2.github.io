df -h 에서 알 수 있는 정보
==================

요약
--
- 제목: **파일 시스템 디스크 사용량 확인**
- 설명: 디스크가 가득 찼다.

로그에서 바로 읽히는 포인트
---------------
- 루트(/) 사용량 행: ``/dev/mmcblk1p5   40G   26G   13G  68% /``

로그 주요 출력(일부)
------------

.. code-block:: text

  cherry@rhombus:~$ df -h
  Filesystem      Size  Used Avail Use% Mounted on
  /dev/mmcblk1p5   40G   26G   13G  68% /
  none             64K     0   64K   0% /dev
  tmpfs           776M  2.1M  774M   1% /run
  tmpfs           5.0M  4.0K  5.0M   1% /run/lock
  tmpfs           776M   96K  776M   1% /run/user/1000
  efivarfs        192K   39K  149K  21% /sys/firmware/efi/efivars
  /dev/mmcblk1p1   96M   56M   41M  59% /boot/efi
  /dev/mmcblk2p1  4.7G  4.7G     0 100% /media/cherry/Ubuntu 22.04.4 LTS amd64
  /dev/mmcblk2p4   55G  1.6M   52G   1% /media/cherry/writable

확인 포인트(체크리스트)
-------------
- 출력에서 **핵심 컬럼 이름**(헤더)을 먼저 읽는다.
- 숫자/단위(%, bytes, blocks, port 등)를 보고 **무엇을 의미하는지** 말로 설명한다.
- 설정 파일이라면(예: fstab) “무엇을 어디에 어떻게 마운트하는지”를 한 문장으로 요약한다.
