리눅스마스터 1급에서 **프로세스 관리 명령어**는 매우 중요한 출제 영역입니다.
특히 다음 세 명령어가 반복적으로 등장합니다.

* `ps`
* `kill`
* `top`

시험에서는 주로 **출력 해석 + 옵션 의미 + 시그널 이해**를 묻습니다.
아래는 **개념 → 옵션 → 실전 문제 → 시험 함정** 순서로 정리합니다.

---

# 1. 프로세스 개념

프로세스란

```text
실행 중인 프로그램
```

구조

```text
program → process
```

각 프로세스는 다음 정보를 가집니다.

```text
PID
PPID
UID
CPU 사용량
메모리 사용량
상태
```

---

# 2. ps 명령어

## 역할

현재 실행 중인 **프로세스 상태 출력**

기본 구조

```bash
ps [옵션]
```

---

# 주요 옵션

| 옵션 | 의미          |
| -- | ----------- |
| -e | 모든 프로세스     |
| -f | full format |
| -u | 특정 사용자      |
| -l | 상세 정보       |
| -a | 터미널 프로세스    |

---

# 자주 사용되는 형태

```bash
ps -ef
```

또는

```bash
ps aux
```

---

# 출력 예

```text
UID  PID  PPID  C STIME TTY TIME CMD
root  1     0   0 00:00 ?   00:00 init
```

---

# 주요 필드

| 필드   | 의미      |
| ---- | ------- |
| PID  | 프로세스 ID |
| PPID | 부모 프로세스 |
| UID  | 사용자     |
| CMD  | 실행 명령   |

---

# 3. 프로세스 상태

시험에서 자주 등장합니다.

| 상태 | 의미                    |
| -- | --------------------- |
| R  | Running               |
| S  | Sleeping              |
| D  | Uninterruptible sleep |
| Z  | Zombie                |
| T  | Stopped               |

---

# Zombie 프로세스

정의

```text
부모 프로세스가 종료 상태를 회수하지 않은 프로세스
```

표기

```text
Z
```

---

# 4. kill 명령어

## 역할

프로세스에 **시그널 전달**

구조

```bash
kill [signal] PID
```

---

# 기본 동작

```bash
kill PID
```

실제

```text
SIGTERM
```

---

# 주요 시그널

| 시그널     | 번호 | 의미    |
| ------- | -- | ----- |
| SIGTERM | 15 | 정상 종료 |
| SIGKILL | 9  | 강제 종료 |
| SIGSTOP | 19 | 실행 정지 |
| SIGCONT | 18 | 실행 재개 |

---

# 예

```bash
kill -9 1234
```

→ 강제 종료

---

# 시그널 확인

```bash
kill -l
```

---

# 5. top

## 역할

실시간 프로세스 상태 확인

실행

```bash
top
```

---

# 주요 정보

```text
CPU 사용량
메모리 사용량
프로세스 목록
```

---

# 주요 필드

| 필드      | 의미      |
| ------- | ------- |
| PID     | 프로세스 ID |
| USER    | 사용자     |
| %CPU    | CPU 사용  |
| %MEM    | 메모리 사용  |
| COMMAND | 명령      |

---

# top 내부 명령

| 키 | 기능      |
| - | ------- |
| k | 프로세스 종료 |
| q | 종료      |
| P | CPU 정렬  |
| M | 메모리 정렬  |

---

# 7. 시험 함정

## 함정 1

```bash
kill
```

의미

```text
프로세스를 죽인다
```

❌ 틀림

정확한 의미

```text
시그널 전달
```

---

## 함정 2

```text
SIGKILL = 정상 종료
```

❌ 틀림

정상 종료

```text
SIGTERM
```

---

## 함정 3

Zombie 프로세스

```text
실행 중
```

❌ 틀림

```text
이미 종료됨
```

---

# 8. 시험 직전 암기

프로세스 상태

```text
R = running
S = sleep
Z = zombie
T = stop
```

---

시그널

```text
SIGTERM = 15
SIGKILL = 9
SIGSTOP = 19
SIGCONT = 18
```

---

# 9. 실제 시스템 흐름

프로세스 확인

```bash
ps -ef
```

문제 프로세스 확인

```bash
top
```

종료

```bash
kill PID
```

강제 종료

```bash
kill -9 PID
```

---

# 핵심 요약

```text
ps = 프로세스 목록
kill = 시그널 전달
top = 실시간 프로세스
```
