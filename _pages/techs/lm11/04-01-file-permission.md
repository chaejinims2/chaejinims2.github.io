리눅스마스터 1급에서 **파일 권한(permission)** 문제는 매우 높은 빈도로 출제됩니다.
문제 유형은 주로 다음 네 가지입니다.

1️⃣ **권한 의미 해석**
2️⃣ **8진수 권한 계산**
3️⃣ **chmod 명령어**
4️⃣ **특수 권한 (SUID, SGID, Sticky bit)**

아래는 **구조 → 권한 계산 → chmod → 특수 권한 → 시험 문제 패턴** 순서로 정리합니다.

---

# 1. 파일 권한 구조

리눅스 파일 권한은 다음과 같이 표시됩니다.

```id="r4h0hm"
-rwxr-xr--
```

구조

```id="r9y2j9"
[파일타입][Owner][Group][Others]
```

예

```id="8vggk4"
-rwx r-x r--
```

| 영역     | 의미     |
| ------ | ------ |
| Owner  | 파일 소유자 |
| Group  | 그룹     |
| Others | 기타 사용자 |

---

# 2. 권한 종류

권한은 세 가지입니다.

```id="9p8f3t"
r = read
w = write
x = execute
```

---

## 파일 권한 의미

| 권한 | 의미    |
| -- | ----- |
| r  | 파일 읽기 |
| w  | 파일 수정 |
| x  | 실행    |

---

## 디렉터리 권한 의미

| 권한 | 의미       |
| -- | -------- |
| r  | 목록 조회    |
| w  | 파일 생성/삭제 |
| x  | 디렉터리 접근  |

시험에서는 **디렉터리 권한 의미**도 자주 물어봅니다.

---

# 3. 8진수 권한 계산

각 권한 값

```id="d1vaz3"
r = 4
w = 2
x = 1
```

---

## 계산 예

### 755

```id="8q4s6g"
7 = 4+2+1 = rwx
5 = 4+1   = r-x
5 = 4+1   = r-x
```

표기

```id="d7d3s4"
rwxr-xr-x
```

---

### 644

```id="a4e7b3"
6 = 4+2 = rw-
4 = r--
4 = r--
```

표기

```id="q8y3m4"
rw-r--r--
```

---

# 4. chmod

권한 변경 명령어

---

## 숫자 방식

```bash id="u9d7k3"
chmod 755 file
```

---

## 문자 방식

```bash id="g3u9a2"
chmod u+x file
```

---

## 문자 의미

| 문자 | 의미     |
| -- | ------ |
| u  | user   |
| g  | group  |
| o  | others |
| a  | all    |

---

## 연산자

| 기호 | 의미 |
| -- | -- |

* | 추가 |

- | 제거 |
  = | 설정 |

---

## 예

```bash id="r5g0az"
chmod g+w file
```

→ group write 추가

---

# 5. 소유권 변경

## chown

소유자 변경

```bash id="wz4kq3"
chown user file
```

---

## 그룹 변경

```bash id="p4g7w3"
chown user:group file
```

---

## chgrp

그룹만 변경

```bash id="k5r8d1"
chgrp group file
```

---

# 6. 특수 권한

시험에서 반드시 등장하는 부분입니다.

---

# SUID

특징

```id="rt9q2s"
실행 시 파일 소유자 권한으로 실행
```

표기

```id="p4r7zs"
-rwsr-xr-x
```

대표 예

```id="w8c9mk"
/usr/bin/passwd
```

---

# SGID

특징

```id="6d0btr"
실행 시 그룹 권한 사용
```

표기

```id="k3j2o1"
-rwxr-sr-x
```

---

# Sticky Bit

디렉터리에서 사용

특징

```id="s2e7pr"
파일 소유자만 삭제 가능
```

대표

```id="b6d0xy"
/tmp
```

---

# 특수 권한 숫자

| 권한     | 값 |
| ------ | - |
| SUID   | 4 |
| SGID   | 2 |
| Sticky | 1 |

---

## 예

### 4755

```id="c7x9a1"
4 = SUID
755 = 일반 권한
```

---

# 7. ls -l 출력 해석

예

```id="v0p2n4"
-rwxr-xr-- 1 user group 1024 file
```

의미

| 필드 | 의미 |
| -- | -- |

* | 파일 타입 |
  rwx | owner |
  r-x | group |
  r-- | others |

---

# 8. 파일 타입

첫 글자

| 문자 | 의미 |
| -- | -- |

* | 일반 파일 |
  d | 디렉터리 |
  l | 심볼릭 링크 |
  c | 문자 장치 |
  b | 블록 장치 |

# 11. 시험 직전 암기

이 다섯 개만 기억하면 됩니다.

```id="d0k8p2"
r=4
w=2
x=1

755 = rwxr-xr-x
644 = rw-r--r--

SUID = 4
SGID = 2
Sticky = 1
```

---

# 핵심 요약

```id="h3b5n9"
권한 = r w x
숫자 = 4 2 1
chmod = 권한 변경
SUID = owner 권한 실행
```

---

원하시면 다음 단계로 **리눅스마스터 1급에서 가장 많이 틀리는**

* **권한 계산 문제 20개**
* **chmod 문제**
* **SUID/SGID 실전 문제**

를 **시험 스타일로 풀어볼 수 있는 문제 세트**도 만들어 드리겠습니다.
