리눅스마스터 1급에서 **SUID / SGID / Sticky bit** 문제는 대부분
① **권한 문자열 해석**
② **숫자 권한 계산**
③ **의미 이해**
형태로 출제됩니다.

아래는 **실전 시험 스타일 문제 → 풀이 → 핵심 정리** 방식으로 구성했습니다.

---

# 문제 1

다음 파일 권한이 의미하는 것은?

```text
-rwsr-xr-x
```

### 보기

1. 실행 시 그룹 권한으로 실행된다
2. 실행 시 파일 소유자 권한으로 실행된다
3. 파일 삭제는 소유자만 가능하다
4. 파일 실행이 불가능하다

### 정답

**② 실행 시 파일 소유자 권한으로 실행된다**

### 해설

권한

```text
rws
```

→ `s` 가 owner 실행 위치에 있음

즉

```text
SUID
```

의미

```text
프로세스 실행 시 owner 권한 사용
```

# 문제 2

```text

```

### 풀이

일반 권한

```text
rwx = 7

```

```text
4
```

따라서

```text
4755
```

### 정답

**4755**

# 문제 8

다음 명령어의 의미는?

```bash
chmod g+s dir
```

### 정답

**디렉터리에 SGID 설정**

### 해설

```text
새 파일이 디렉터리 그룹 상속
```

# 시험 핵심 패턴

시험에서는 다음 형태가 가장 많이 나옵니다.

### 1️⃣ 문자열 해석

```text
-rwsr-xr-x
```

→ SUID

---

### 2️⃣ 숫자 계산

```text
4755
2755
1777
```

---

### 3️⃣ 의미 문제

| 권한     | 의미          |
| ------ | ----------- |
| SUID   | owner 권한 실행 |
| SGID   | group 권한 실행 |
| Sticky | 디렉터리 삭제 보호  |

---

# 시험 암기

다음 세 개만 정확히 기억하면 됩니다.

```text
SUID   = 4
SGID   = 2
Sticky = 1
```

---

# 실제 시스템 예

대표적인 SUID 프로그램

```text
/usr/bin/passwd
```

권한

```text
-rwsr-xr-x
```

이유

```text
일반 사용자가 비밀번호 변경 가능
```
